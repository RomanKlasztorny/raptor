# РАПТОР — конструктор архитектуры

Браузерный инструмент для проектирования и документирования архитектуры систем.

**[→ Открыть приложение](https://romartamonov.github.io/raptr/)**

## Возможности

- Визуальное проектирование в нотации C4 (C1/C2/C3/Sequence)
- Симуляция нагрузки и трафика между сервисами
- Живой инспектор брокеров (Kafka, RabbitMQ, NATS)
- Редактор схем БД с визуальным ERD
- Экспорт документации в Word (формат MS-1)
- Шаблоны: E-commerce, Fintech, Сага, CQRS, SSO и другие

## Стек

Чистый HTML + JS, без сборки и зависимостей. Открывается в браузере.

## Локальный запуск

```bash
npx serve .
```
