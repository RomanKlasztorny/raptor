🏗 Как пользоваться

Палитра + холстВыбери тип слева → кликни на холст. Наведи на тип — подсказка где ставить и где нельзя
Клик на блокСправа: что это, живая нагрузка %, задержка, где ставить, советы
Тянуть от ○Навести → белый кружок справа → тянуть к другому блоку. Появится совет правильно/нет
2× клик блокНастройки: реплики, PgBouncer, партиции. 2× клик на брокере в симуляции → инспектор
Клик стрелкуЧто несёт, кто кого зовёт, нагрузка, API. Двунаправленные стрелки = запрос+ответ
C1 / C2 / C3 / C4C4 уровни: Context → Container → Component → Dynamic (sequence)
📐 АвтоАвторасстановка по слоям: клиенты → gateway → сервисы → брокеры → хранилища
🗄 БД / 🔐 AuthШаблоны баз данных (20 штук) и авторизации (3 шаблона) с готовыми схемами и SQL
Двунаправленные стрелки: Клиент↔Gateway, Gateway↔Сервис, Сервис↔БД — стрелки с двух сторон.
Kafka: Сервис может быть одновременно продюсером (→Kafka) и консьюмером (Kafka→) разных топиков.
Бизнес-язык в UML: Над стрелкой — что делает (бизнес). После // — как именно (API).
Коллизия: При перетаскивании блока на другой — красная рамка и откат назад.
